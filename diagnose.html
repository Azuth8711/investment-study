解决方案

如果本地文件系统加载失败，可能的解决方案：

  1. 使用本地HTTP服务器：在文件夹中运行 python -m http.server 8000
  2. 使用浏览器扩展：安装允许本地文件CORS的扩展
  3. 修改浏览器设置：启动Chrome时添加 --allow-file-access-from-files 参数
  4. 使用WorkBuddy产物列表：它可能已经通过HTTP服务器提供服务
